## Git Strategy

### 전체 브랜치
- master
- develop
- release
- feature/...
- bugfix/...
- hotfix/...

### origin/master
운영 브랜치

### origin/develop
개발 브랜치

### origin/release
스테이지 브랜치

### 현재 버전에서 수정 사항 반영 시

```
                                  (new branch)      (merge request)   
- master                      ─────o────────────────o─────────────────────
- hotfix/fix-blah-blah-error  	   └───o───o───o────┘
```
